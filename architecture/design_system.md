# Strategic Dashboard Design System (Revamp v3)

Based on analyzed inspiration:
1. [News Inspo](file:///c:/Users/micha/Aleph%20Dashboard%20V2/Inspo/inspo_1.png) - Grid-based exploration and channel branding.
2. [Info Flow Inspo](file:///c:/Users/micha/Aleph%20Dashboard%20V2/Inspo/inspo_2.png) - Saturated gradients and clean modular cards.
3. [Analytics Inspo](file:///c:/Users/micha/Aleph%20Dashboard%20V2/Inspo/inspo_3.png) - Bold secondary color blocks for data categorization.

## 🎨 Color Palette
- **Canvas:** `#0c0d10` (Deep Space)
- **Base Card:** `#16181d`
- **Surface High:** `#24262b`
- **Accents:**
    - **Reliability (Outages):** `#ff3b3b` (Vivid Red)
    - **Velocity (Launches):** `#00a3ff` (Electric Blue)
    - **Capital (Funding):** `#00ffa3` (Neon Green)
    - **Policy (Regulation):** `#ffd600` (Vivid Yellow)
- **Gradient Start (Synthesis):** `linear-gradient(135deg, #00a3ff 0%, #ff0080 100%)`

## 📐 Components
- **Global Sidebar:** Narrow, persistent navigation with region/vertical filters.
- **Synthesis Banner:** Hero-style card for the "Market Momentum" thesis.
- **Signal Cards:**
    - **Standard:** Horizontal layout with category badge.
    - **Featured:** Vertical layout with image support (scraped from source).
- **Status Pills:** High-contrast rounded pills for live health checks.

## 🔡 Typography
- **Headings:** `Inter`, Semi-Bold, 1.25rem, Letter spacing 1px.
- **Data/Mono:** `IBM Plex Mono`, Medium for technical alerts.
- **Body:** `Inter`, Regular, 0.9rem, high line-height (1.6).
